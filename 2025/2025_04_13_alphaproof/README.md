# Olympiad-level formal mathematical reasoning with reinforcement learning

**Presenter:** Bartosz Naskręcki  
**Slides:** [2025_04_13_alphaproof.pdf](./2025_04_13_alphaproof.pdf)  
**Paper:** Hubert et al., [Olympiad-level formal mathematical reasoning with reinforcement learning](https://doi.org/10.1038/s41586-025-09833-y)  
**Venue:** *Nature* **651**, 607-613 (19 March 2026)

This seminar presents AlphaProof, DeepMind's reinforcement-learning system for
formal theorem proving in Lean. The talk starts from lambda calculus and
Curry-Howard, explains the Lean proving environment, proof network, PUCT
search, auto-formalization, main reinforcement learning, and test-time
reinforcement learning, and ends with a fully verified theorem plus an explicit
view of the underlying proof terms and local algebra certificates.
